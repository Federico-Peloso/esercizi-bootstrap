Risposte
1) Bootstrap è un framework che permette di velocizzare lo sviluppo front-end di un sito web, quindi fornisce delle classi pre impostate che permettono di stilizzare gli elementi di una pagina in modo più veloce.
2) Per usufruire di bootstrap è necessario includere il file css di bootstrap dal server (quindi con il cdn): <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">.
3) Un cdn permette di includere il contenuto di un file presente su un server all'interno del proprio porgetto.
4) L'attributo content="width=device-width, initial-scale=1 imposta la larghezza in pixel della pagina in modo che sia uguale alla larghezza in pixel del dispositivo su cui viene visualizzata, oltre a impostare il livello di zoom iniziale come predefinito.
